API Reference

Authentication

Use API keys to authenticate requests. Your API keys carry many privileges, so be sure to keep them secure!

Authorization: Bearer YOUR_API_KEY 
curl -X POST https://api.example.com/v1/data \
-H "Authorization: Bearer sk_test_123" \
-d "param=value"

Endpoints

GET /v1/users

Returns a list of users.

200 OK

POST /v1/users

Creates a new user.

201 Created
